eCommerce Platform

Webstore

Role: Full Stack Engineer

Stack: Rails, JavaScript, jQuery, PostgreSQL, Ruby, AJAX, Stripe, BCrypt, Paperclip

The client requested a shopping website similar to Amazon. The eCommerce platform included an inventory management section for admins. User authentication was also required, as well as automated confirmation emails.

The webstore was built with Rails. AJAX and jQuery was used to create a dynamic user experience. The site was designed to be responsive and mobile friendly.
