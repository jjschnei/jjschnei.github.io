Randeaux

Ruby Gem

Role: Ruby Developer

Stack: Ruby

A Ruby gem that returns randomized data from arrays and hashes.

The randeaux methods takes two arguments: the data structure you wish to sample and an optional argument for the number of elements you wish to be returned. By default, randeaux will return one random element.

The randeaux_index method takes the same arguments, except it returns random indexes instead of random values.
