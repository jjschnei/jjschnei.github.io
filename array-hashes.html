The Basics of Ruby Hashes and Arrays

December 6th, 2015
Jeff Schneider

As you begin to explore Ruby, you will find yourself needing to organize not just individual objects, but collections of objects. Ruby allows you to do this with container objects. Ruby comes with two built-in container objects: meet the Array and Hash!

So What Exactly is an Array?

An array is an ordered collection of objects. We say that it’s ordered because each object is indexed: 

       array_name = [a,  b,  c,  d,  e]
                     |   |   |   |   |
       Index number  0   1   2   3   4
(Note that the first object in an array has 0)

Here is the syntax to look up a particular object in the above array called array_name: 
        Array_name .index [2]    => c
You can put any object inside of an array such as: integers, strings, true/false, and even other arrays. To look up the last object in an array, use index -1.

There are four ways to create a new array:

  • With the Array.new method
  • With the literal constructor
  • With a top level method called Array
  • With the special %w{. . .} and %i{. . .} notations

So What Exactly is an Array?

Hashes store ordered objects in pairs called keys and values. You can store any object as a key or value. You use the key to call its value. Each key must be unique, though values can be repeated. Hashes are ordered by the order in which you entered key/value pairs. Like arrays, hashes are ordered objects because they remember the order in which they were entered.
There are four ways to create a new hash:

  • With the literal constructor
  • With the Hash.new method
  • With the Hash.[] method
  • With the top-level method whose name is Hash

Here is an example of creating a new state capitals hash using the Hash.new method: 

          state_capitals = Hash.new
Once we’ve created our new hash, it’s time to fill it with key/value pairs. There are several ways you can add key/value pairs. Here are some examples of the different ways to do it: 
          state_capitals [“Alabama”] = “Montgomery”
          state_capitals.store (“Alaska”, “Juneau”)
          state_capitals [] = (“Arizona”, “Phoenix”)
To retrieve a value from a hash, you use the [] method. Much like calling an object from an array using the index number, you can access values using a key.
