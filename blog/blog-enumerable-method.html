December 14th, 2015
Jeff Schneider

What's an Enumerable?

One of the things that makes Ruby powerful is being able to use Enumerable methods. “Enumerable” is a “module”. Simply stated, that means it a bunch of methods packaged together that can be “mixed in” with other classes (like Arrays and Hashes).

This means that you don’t have to write out every method a bunch of times. You can write them just once, package them up as Enumerable, and then tell Array and Hash to use them. The only catch is that the class on which you want to use Enumerable has to have its own #each method that Enumerable can use.

Let's take a look at how Enumerable provides useful ways of doing things to elements of a collection object (like an Array or Hash). For example, you are probably familiar with the #each iterator method that comes pre-packaged with the Array and Hash classes.
